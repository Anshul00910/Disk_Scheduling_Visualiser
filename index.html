What is Disk Scheduling?

Disk scheduling is the method operating systems use to decide which pending I/O requests should be serviced next. Efficient disk scheduling algorithms minimize seek time, optimize throughput, and reduce latency in data access operations.

Why It Matters

Over 20% of system time can be optimized with proper disk scheduling! Even in modern SSDs, efficient scheduling algorithms can significantly improve system performance and response times.

Algorithm Explorer

FCFS

FCFS

Requests are addressed in the order they arrive without any reordering. Simple but often inefficient for disk operations.
SSTF

SSTF

Always services the request closest to the current head position. Can cause starvation of some requests.
SCAN

SCAN

The disk arm moves in one direction servicing requests until it reaches the end, then reverses direction.
C-SCAN

Circular SCAN

Like SCAN, but when reaching the end, the arm returns to the beginning without servicing requests on the way back.
LOOK

LOOK

Similar to SCAN but the arm only goes as far as the last request in each direction, then reverses.
CLOOK

C-LOOK

Like C-SCAN but the arm only goes as far as the last request in each direction before jumping.
