About SSTF Disk Scheduling

SSTF (Shortest Seek Time First) is a disk scheduling algorithm where the request that is closest to the current head position is selected next. This approach aims to minimize the seek time by always selecting the closest request, but it can result in starvation if there are frequent requests at one end.
